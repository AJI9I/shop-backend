# Остановка приложения перед сборкой

Если вы получаете ошибку "файл занят другим процессом" при сборке, нужно:

1. **Остановить Spring Boot приложение**, если оно запущено:
   - В IDE: нажмите кнопку "Stop" на панели запуска
   - В терминале: нажмите `Ctrl+C` в окне, где запущено приложение
   - Через диспетчер задач: найдите процесс `java.exe` и завершите его

2. **Очистить директорию target**:
   ```bash
   cd shop-backend
   mvn clean
   ```

3. **Пересобрать проект**:
   ```bash
   mvn compile
   ```

4. **Если проблема сохраняется**, попробуйте:
   - Закрыть IDE и открыть снова
   - Перезапустить компьютер
   - Удалить директорию `target` вручную и пересобрать
